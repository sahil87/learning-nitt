Instructions to include JavaScript Matrix Background to your own pages:

1. Copy usingit_js_matrix_background_cfg.js and usingit_js_matrix_background.js to the same directory along with the html file which you want to include this script.
* You may copy the JavaScript files to different directory if you know how to set the path of them in html files.

2. Copy all jpg files (m1.jpg - m10.jpg, m1b.jpg - m10b.jpg) to the same directory, see above.
* You may copy jpg files to different directory if you know how to modify JavaScript codes.

3. Copy

<script language="javascript" type="text/javascript" src="usingit_js_matrix_background_cfg.js"></script>
<script language="javascript" type="text/javascript" src="usingit_js_matrix_background.js"></script>

to head section of the html files.

4. You may need to set the background color of the page to Black (#000000).

5. You may need to set the z-index of all web content within the page to 1 or higher to display the matrix animation behind them. In default, I set the matrix animation's z-index to -1000.

6. Finally, if you need to adjust this script, please open usingit_js_matrix_background_cfg.js from any text editor. There are 2 parameters. See details within that file.
